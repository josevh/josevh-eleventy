# josevh.com

My personal website made with Eleventy.

## Stack
- Eleventy v3
- Nunjucks/Liquid/Markdown templates
- Sass (`dart-sass`) compiled via `.eleventy.js`

## Quick Start
```bash
npm install
npm run serve
```

## Build
```bash
npm run build
```

Static output is generated in `dist/` (do not edit `dist/` manually).

## Project Structure
- `src/` pages and templates
- `src/blog/` blog posts
- `src/projects/` project pages
- `src/_data/site.json` site metadata
- `src/_data/site/experience.json` and `src/_data/site/education.json` resume data

## Language
Single-language site: `en-US`.
